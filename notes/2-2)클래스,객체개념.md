## Class & Object

### **1) 클래스란?**

- **객체를 만들기 위한 설계도(틀)**
- 같은 속성/기능을 가진 대상들을 **하나로 묶어 정의**할 때 사용

### **2) 객체란?**

- 클래스로부터 만들어진 **실제 개체(인스턴스)**
- 예) 클래스가 Person이면 객체는 유재석, 박명수처럼 각각 **서로 다른 존재**

---

### **3) 객체 만들기**

```python
class Person:
    pass

person_1 = Person()
person_2 = Person()
```

- Person()을 호출하면 **새 객체(인스턴스)** 가 생성됨

---

### **4) 생성자(Constructor)**

### **__init__**

- 객체가 생성될 때 **자동으로 실행되는 함수**
- 파이썬 생성자 이름은 **무조건 __init__**

```python
class Person:
    def __init__(self):
        print("hihihi", self)
```

### **5) self 의미**

- self는 **객체 자기 자신**
- 따로 넣지 않아도 메서드 호출 시 파이썬이 자동으로 전달해줌

---

### **6) 객체에 데이터 저장 (속성/필드)**

- self.name = 값 형태로 **객체 내부에 값 저장**

```python
class Person:
    def __init__(self, param_name):
        self.name = param_name

person_1 = Person("유재석")
print(person_1.name)  # 유재석
```

### **7) 메서드(Method)**

- 클래스 안에 정의된 함수 = **메서드**
- 객체의 데이터(self.name)를 활용할 수 있음

```python
class Person:
    def __init__(self, param_name):
        self.name = param_name

    def talk(self):
        print("안녕하세요 저는", self.name, "입니다")

person_1 = Person("유재석")
person_1.talk()
```

---

### **한 줄 결론**

- **클래스는 설계도**, **객체는 그 설계도로 만들어진 실제 결과물**이고,
    
    __init__과 self를 통해 **객체별 데이터와 기능을 묶어서 관리**할 수 있다.
